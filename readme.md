 ### 2.เล่รเรื่องอะไรก็ได้เกี่ยวกับตัวเอง 3 ข้อ โดย 2 ข้อเป็นเรื่องจริง 1 ข้อ เป็นเรื่องโกหก..
> Ans:  
> ผมเรียนได้ที่ 1 ของห้อง (false)  
> ผมมีพี่สาว 1 คนและน้องชาย 2 คน (false)
